Let's create a High-Load Infra Playground

A local experiment lab for learning production-grade backend infrastructure: Node.js + Postgres + PgBouncer + Redis + RabbitMQ + Prometheus + Grafana.

Goals:
- Learn how to build a production-grade high-load backend infrastructure
- Learn how to use Prometheus and Grafana to monitor and visualize the infrastructure
- Learn how to tune PgBouncer to improve performance
- Learn how to use RabbitMQ for event-driven architecture, where consumers can be slower than producers
- Learn how to use Redis for the same task as RabbitMQ: event-driven architecture where consumers can be slower than producers

Notes: The system in general should be a prediction market with Logarithmic Math Scoring Rules math.
